membrane.js
======================
A javascript library for membrane (p-system) computing and eventually visualizations for them.

# Features
Currently Supported Rules
- Evolution
- Send Out
- Send In
- Dissolve

# Releases
To build a release
- `grunt bump-only:patch`, `grunt bump-only:minor`, or `grunt bump-only:major`
- `grunt test`
- `grunt bump-commit`
